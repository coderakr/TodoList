# TodoList Program

This is a simple TodoList program written in C++.

## Getting Started

### Prerequisites

To compile and run the program, make sure you have a C++ compiler (like `g++`) installed on your system.

### Compilation

To compile the code:

```bash
g++ main.cpp -o todolist
```

### Run (in linux)

To run the code:

```bash
./todolist
```
